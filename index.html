HTML

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje estándar para crear páginas web. No es un lenguaje de programación, sino un lenguaje de marcado, lo que significa que se usa para estructurar y organizar el contenido de una página web mediante etiquetas. ✅ ¿Para qué sirve? Definir la estructura de una página web: encabezados, párrafos, imágenes, enlaces, tablas, formularios, etc. Indicar al navegador cómo mostrar el contenido. ✅ Elementos básicos Un documento HTML tiene etiquetas que van entre < > y suelen venir en pares:

CSS

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje usado para dar estilo y diseño a las páginas web creadas con HTML. Mientras HTML define la estructura (qué elementos hay), CSS define la apariencia (cómo se ven). ✅ ¿Para qué sirve CSS? Cambiar colores, tipografía, márgenes, espaciado. Hacer que la página sea responsiva (se adapte a móviles). Crear animaciones y efectos visuales. Organizar el diseño con grid, flexbox, etc. ✅ ¿Cómo se usa? Hay tres formas: CSS en línea (inline): dentro de la etiqueta HTML.
